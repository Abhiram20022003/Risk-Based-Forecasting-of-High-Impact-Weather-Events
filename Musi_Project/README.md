MUSI RIVER BASIN AI FLOOD FORECASTING PROJECT

How to run:
1. Install Python 3.10 or 3.11 recommended.
2. Double-click INSTALL_REQUIREMENTS.bat
3. Double-click RUN_DASHBOARD.bat
4. Browser opens the Streamlit dashboard.

Main features:
- AI-based rainfall-runoff forecasting prototype
- Future predicted flood-risk map
- Low, Medium, High risk colour classification
- Observed vs predicted discharge graph
- Scatter plot, residual plot, loss curve
- Model comparison graph
- Regime shift analysis
- Download metrics CSV
- Export PDF report
- Generate dissertation-style DOCX summary

Note:
This is a research demonstration dashboard with synthetic/sample Musi Basin data.
You can replace data files in the data folder with real station, rainfall, water level, or GIS data.
